# 7차 진행: PR 품질 게이트 + 완료 기준 고정

이번 단계는 "언제 끝난 것으로 볼지"를 명확히 하기 위해 PR 템플릿 기반 품질 게이트와 완료 기준을 고정한다.

## 적용 항목
- `.github/pull_request_template.md` 추가
  - 웹 점검 체인(`npm run check:web:ci`) 통과 확인
  - include 변경 영향 범위 기록
  - UI 변경 시 수동 스모크 체크 항목 명시

## Done 기준(제안)
아래를 모두 만족하면 구조 안정화 단계 완료로 본다.

1. 최근 2주간 web-check CI 실패 0건
2. 미참조/중복/문법/전역충돌/inline 참조 누락 이슈 0건 유지
3. PR 템플릿 체크리스트 누락 없이 운영
4. 남은 구조 개편은 "품질 게이트 강화"가 아닌 "아키텍처 선택(ESM 전환 등)"으로 전환

## 예상 남은 단계(현 시점)
- 최소 1~2단계:
  - (선택) CI 결과 요약 리포트 자동 코멘트
  - (선택) ESM 전환 PoC 혹은 include 매니페스트화

즉, 현재까지의 안정화 트랙은 거의 마무리 단계다.
